HTML y CSS.

Guía de ejercicios.


  1. Indicar verdadero o falso.
    • Los elementos de HTML solo pueden tener un atributo. F
    • El modelo de caja de CSS especifica que cada elemento tiene un margen, un relleno y un contenido. F
    • HTML significa lenguaje marcado de hipertexto. V
    • Existen elementos en HTML que no tienen atributos. F
    • En CSS un selector que empiece con . indica una clase. V
    • El elemento img es un elemento vacio. V
    • Lo que está dentro de la etiqueta head es visible por el usuario. F
    • En CSS, el selector universal se denota con un asterisco. V
    • Un span es un elemento en el bloque. F
    • En CSS, cuando 2 selectores de igual especificidad se aplican al mismo elemento, prevalece la regla que aparece primero en la hoja de estilos. F
    • El atributo href de un anchor es una referencia. V
    • Una lista ordenada en HTML se construye con un elemento ol y tantos elementos li como elementos en la lista. V
    • Todos los elementos en HTML pueden tener los atributos src y class. F
    • En CSS podemos cambiar el espacio entre elementos consecutivos usando la propiedad padding. F
    • Toda la página web termina con una etiqueta body. F
    • Los atributos comunes a todas las etiquetas son id y class. V
    • Un em es un elemento en línea. V
    • Todas las etiquetas de HTML vienen en pares, una que abre y otra que cierra. F

  2. Completar las afirmaciones.
    • Lo que está entre el borde y el contenido de un elemento. = padding
    • Nombre del elemento que sirve para crear hipervínculos. = anchor
    • Elemento en bloque que no tiene significado. = div
    • Un elemento para mostrar código fuente. = pre
    • Los elementos h1 a h6 son = headings
    • Similar un checkbox pero solo admite una opción. = radio
    • Una hoja de estilos es un conjunto de = reglas
    • Elemento en línea que no tiene significado = span
    • Celda en el encabezado de una tabla = th
    • En el modelo de caja, el margen que separa un párrafo del párrafo previo es el margin = top

  3. Completar los espacios en blanco.
    • a href="www.google.com" target="_blank" Google /a (abrir el link en una nueva pestaña).
    • El elemento span es a los elementos en línea lo que div es a los elementos en bloque.
    • img src="img/doge.png" alt="un doge"/ (usar como texto alternativo "un doge").
    • Un documento HTML tiene estructura de arbol y su elemento raíz es el elemento html.
    • Para agregar énfasis a una palabra dentro de un párrafo podemos usar un elemento em anidado encerrando la palabra que queremos destacar.
    • En CSS, para seleccionar todos los li dentro de un elemento nav con clase menu usamos nav.menu li {}
    • En CSS para darle sombra al borde de un elemento usamos la propiedad box-shadow.
    • La única unidad de medida absoluta usada comunmente en CSS es el píxel.
    • La propiedad de CSS para cambiar el tamaño de fuente es front-size.
    • Los atributos de los elementos HTML se ponen en la etiqueta que abre el elemento.
    • Para sacar el subrayado de un link se usa la propiedad text-decoration en CSS, con el valor none.
